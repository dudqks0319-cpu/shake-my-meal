# ShakeMyMeal Store Metadata Draft

## App Name

- 흔들밥

## Subtitle

- 흔들어서 오늘 메뉴 결정

## Short Description

- 폰을 흔들면 오늘 먹을 메뉴를 오프라인으로 바로 추천해주는 앱

## Promotional Text

- 점심 메뉴 고민을 5초 안에 끝내세요. 살살 흔들면 가볍게, 세게 흔들면 푸짐하게 추천해줍니다.

## Keywords

- 점심메뉴,메뉴추천,랜덤추천,뭐먹지,흔들기,데이트,친구,식사

## Privacy Summary

- 로그인 없음
- 서버 저장 없음
- 메뉴 추천과 설정은 기기 내부에 저장
- 센서는 흔들기 강도 측정에만 사용

## Version 1 Scope

- 오프라인 메뉴 추천
- 설정과 기록 저장
- 같은 화면 배틀
- 시스템 공유

## Excluded From Version 1

- 블루투스 배틀
- 결제
- 광고
- 전용 SNS SDK 공유
